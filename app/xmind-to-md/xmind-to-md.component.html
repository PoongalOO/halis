Convertisseur XMind vers Markdown

Procédure à suivre

Comprendre le format XMind : Un fichier XMind est essentiellement une archive (format ZIP) qui contient des fichiers XML ou JSON décrivant la structure de la carte mentale. Vous pouvez renommer l'extension en .zip et l'extraire pour accéder à son contenu.

  • Extraire le contenu : Une fois extrait, repérez le fichier content.json qui contient la structure principale de la carte mentale.
  • Lecture du fichier : Sélectionnez le fichier content.json via le formulaire. Le FileReader lit le contenu du fichier, et le JSON est ensuite parsé.
  • Extraction du rootTopic : Le fichier content.json est lu puis affiché à l'écran au format md.
  • Téléchargement du Markdown : Une fois convertit, vous pouvez télécharger le fichier au format md, format compréhensible et utilisable par HALIS pour d'autre forme d'affichage.
